Stay informed with real-time, AI-powered blog posts curated from reliable news sources across the web.

About This App

This blog application fetches and displays the latest trending articles based on social engagement metrics. Whether you're a news enthusiast or just looking to stay updated, this platform brings top stories to your fingertips.

Key Features

  • 🔥 Fetches real-time blog posts using social score algorithms
  • 🖼️ Includes images with consistent sizing and fallback support
  • 🔗 Direct source links for every article
  • 📅 Displays human-readable publish dates
  • 🚀 Responsive and clean user interface using Bootstrap

Technologies Used

  • Python & Django
  • BeautifulSoup / News APIs
  • SQLite / PostgreSQL
  • HTML5 & Bootstrap 5
  • Font Awesome
  • Template Inheritance (Django Templates)
